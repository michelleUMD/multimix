# multimix

**Author:** Michelle Fang\
**Date:** 2026-01-15

`multimix` is an R package for fitting and visualizing the *Biphasic Temporal Decomposition Logistic Mixed Effects Model*

------------------------------------------------------------------------

## Installation

You can install the development version from GitHub:

``` r
install.packages("devtools") 
devtools::install_github("michelleUMD/multimix")
```
